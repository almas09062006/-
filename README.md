# AI Legal Document Analyzer

Учебный веб-проект для анализа нормативных, служебных и корпоративных документов.

## Возможности

- загрузка TXT-файла;
- вставка текста документа вручную;
- формирование краткого резюме;
- поиск сроков и дат;
- поиск рисков и обязательств;
- вопрос по документу;
- сохранение результата анализа в истории;
- скачивание результата в формате TXT;
- скачивание исходного текста.

## Стек

- HTML
- CSS
- JavaScript
- LocalStorage

## Как запустить

1. Скачайте проект.
2. Откройте файл `index.html` в браузере.
3. Загрузите TXT-файл или вставьте текст документа.
4. Нажмите нужную кнопку анализа.

## Что можно добавить позже

- backend на Python Flask или Node.js;
- загрузку PDF и DOCX;
- подключение OpenAI API;
- экспорт результата в DOCX/PDF;
- авторизацию пользователей;
- хранение файлов в базе данных.

## Назначение проекта

Проект может использоваться как прототип AI-системы для анализа нормативных документов, приказов, положений и официальных писем.
